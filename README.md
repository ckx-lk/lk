# 广东兰凯律师事务所官方网站

GUANGDONG LANKAI LAW FIRM

## 项目说明

本仓库为广东兰凯律师事务所官方网站源代码，包含前端展示页面与后台管理系统。

## 技术栈

- **前端**：HTML5 / CSS3 / JavaScript
- **后台**：Classic ASP + Access/MSSQL
- **主色调**：清华大学紫色系 (#4B0082)

## 目录结构

```
├── index.html          # 首页
├── about.html          # 关于我们
├── practice.html       # 业务领域
├── team.html           # 专业团队
├── news.html           # 新闻中心
├── research.html       # 兰凯研究
├── party.html          # 兰凯党建
├── css/                # 样式文件
├── js/                 # 脚本文件
├── images/             # 图片资源
├── admin/              # 后台管理系统（HTML预览版）
├── asp/                # ASP版本代码
└── data.json           # 结构化数据
```
